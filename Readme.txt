Para poder utilizar la aplicación se debe modificar el connectionString

Crear Productos

- Una vez que la aplicación este corriendo se debe crear un Supplier y Category previamente. Después se procede a crear el Producto. una vez creado el producto podemos interactuar
con la aplicación.

Comprar

- Para poder realizar una compra se debe seleccionar el producto y después dar click en el botón de add to cart esto nos redirecciona a la vista de Shopping Cart y para que se pueda
realizar la compra exitosamente el usuario debe estar registrado y con la sesion iniciada en la aplicacion.
